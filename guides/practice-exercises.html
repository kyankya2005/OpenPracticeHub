üéâ Exercise 1: Your First Contribution

Goal: Make your very first open-source contribution

Time: 5-10 minutes

Skills: Basic Git workflow, PR creation

Steps:

  1. Fork OpenPracticeHub
  2. Clone your fork
  3. Create branch: git checkout -b add-my-first-quote
  4. Open projects/random-quotes/quotes.json
  5. Add your favorite motivational quote:
{
  "text": "Your favorite quote here",
  "author": "Author name"
}
  1. Save the file
  2. Stage: git add .
  3. Commit: git commit -m "Add motivational quote by [Author]"
  4. Push: git push origin add-my-first-quote
  5. Create PR on GitHub!

What you learned: Basic Git workflow, JSON editing, PR creation

üîç Exercise 2: Fix a Typo

Goal: Practice finding and fixing small issues

Time: 3-5 minutes

Skills: File editing, committing

Steps:

  1. Read through the README.md file
  2. Look for any typos or grammar issues
  3. Create branch: git checkout -b fix-typo
  4. Fix the typo(s)
  5. Commit: git commit -m "Fix typo in README"
  6. Push and create PR!

What you learned: Attention to detail, quick fixes, documentation improvement

üÜï Exercise 4: Add a New Feature

Goal: Build something new from scratch

Time: 20-30 minutes

Skills: HTML, CSS, JavaScript, problem-solving

Feature ideas:

  • Dark mode toggle: Add a button to switch between light/dark themes
  • Search functionality: Let users search through projects
  • Back-to-top button: Smooth scroll to top
  • Loading animations: Make waiting feel better
  • Social sharing buttons: Share projects on social media

Steps:

  1. Plan your feature (what will it do?)
  2. Create branch: git checkout -b add-[feature-name]
  3. Build your feature
  4. Test it works
  5. Commit with clear message
  6. Push and create PR!

What you learned: Feature development, testing, user experience
